1. Create venv
    `python -m venv my-venv`

2. Activate venv
    `source my-venv/bin/activate`
    - Note, need to do this every time you want to use venv 

3. Install deps (requirements.txt)
    `python -m pip install -r requirements.txt`

4. Check deps
    `python -m pip freeze`

5. Deactivate venv
    `deactivate`
